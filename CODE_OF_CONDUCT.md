# Code of Conduct

## Our commitment

We are committed to a welcoming, harassment-free community for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and collaborate in good faith.
- Give and receive feedback professionally.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Trolling, insults, or inflammatory comments
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, and other contributions that violate these standards.

## Reporting

Please report unacceptable behavior through GitHub Issues or by contacting maintainers through repository channels.
