# Security Policy

## Supported versions

Security updates are provided for the latest code on the default branch.

## Reporting a vulnerability

If you discover a security issue, please open a private security advisory on GitHub for this repository.

When reporting, include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested remediation (if available)

Please avoid publicly disclosing vulnerabilities until maintainers confirm and patch the issue.
